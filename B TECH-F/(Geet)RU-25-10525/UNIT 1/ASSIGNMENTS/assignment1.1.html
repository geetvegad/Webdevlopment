HTML Attributes

1) What are Attributes?

Attributes provide extra information inside an HTML element, composed of name and value pairs like name="value".

2) Example — <a> (href, target, rel)

The href attribute specifies link destination, target controls where it opens, and rel defines relationship/security.

Example: Visit Wikipedia

Code:

<a href="https://www.wikipedia.org" target="_blank" rel="noopener noreferrer" title="Visit Wikipedia Website">Visit Wikipedia</a>

3) Example — <img> (src, alt, width, height)

src defines image source, alt shows text if image fails, and width, height set image size.

Example:

Code:

<img src="nayu.jpg" alt=" Image" width="250" height="150">

4) Some Other Common Attributes

  • class — Used to assign CSS or JS class groups.
  • id — Uniquely identifies an element on the page.
  • title — Displays a tooltip on mouse hover.
  • style — Apply CSS styles directly to an element inline.
